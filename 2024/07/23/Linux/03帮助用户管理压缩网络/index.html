# 帮助命令

# man

获得帮助信息

原意：

  • manual

执行权限：

  • 所有用户

语法：

 

# whatis

显示命令的简要描述

执行权限：

  • 所有用户

语法：

 

# apropos

显示配置的简要描述

执行权限：

  • 所有用户

语法：

 

# info

读取 info 信息

执行权限：

  • 所有用户

语法：

 

# help

获得 Shell 内置命令的帮助信息

命令所在路径：

  • Shell 内置命令

    使用 which 或其他查找命令找不到信息 的命令即为 Shell 内置命令

执行权限：

  • 所有用户

语法：

--help（非命令）

语法： [命令] --help

执行权限：

  • 所有用户

语法：

 

# w

执行权限：

  • 所有用户

# 压缩解压命令

# gzip

压缩文件至 .gz 格式（不能压缩目录）

原意：

  • GUN zip

执行权限：

  • 所有用户

语法：

 

# gunzip

解压缩 .gz 的压缩文件

原意：

  • GUN unzip

执行权限：

  • 所有用户

语法：

 

# zip

压缩文件或目录至 .zip 格式

执行权限：

  • 所有用户

语法：

 

# unzip

解压 .zip 的压缩文件

执行权限：

  • 所有用户

语法：

 

# bzip2

压缩文件或目录至 .bz2 格式

执行权限：

  • 所有用户

语法： c

 

# bunzip2

解压缩

执行权限：

  • 所有用户

语法： c

# write

  • 所有用户

# wall

  • write all

  • 所有用户

语法：

 

# ifconfig

原意：

  • interface configure
